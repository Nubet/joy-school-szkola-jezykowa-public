# Joy School - Szkoła Języków Obcych

Projekt strony internetowej dla szkoły językowej Joy School, zbudowany w `Next.js` z wykorzystaniem `React`,
`TypeScript` i `Tailwind CSS`.

Szkoła wcześniej nie posiadała własnej witryny - strona powstała od zera. Skupia się na tym, co realnie wyróżnia
ofertę: promocji półkolonii oraz przejrzystym, czytelnym cenniku.

## Co zostało zrobione

- **zbudowanie strony od zera** - szkoła wcześniej nie posiadała własnej witryny
- **promowanie półkolonii** - wyeksponowane jako istotny element oferty
- **przejrzysty cennik** - czytelna prezentacja cen, ułatwiająca rodzicom podjęcie decyzji
- uporządkowanie oferty i najważniejszych informacji dla rodziców i kursantów
- spójna stylistyka i czytelny układ strony
- nowoczesna, wiarygodna prezentacja szkoły

## Efekt

Powstała krótka, czytelna i przekonująca strona, która porządkuje komunikację szkoły i ułatwia szybkie przejście do
kontaktu.

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:3000`.

## Skrypty

- `npm run dev` - uruchamia środowisko developerskie
- `npm run build` - buduje aplikację produkcyjną
- `npm run start` - uruchamia build produkcyjny
- `npm run lint` - uruchamia ESLint
